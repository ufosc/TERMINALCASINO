# Get started

Welcome to TERMINALCASINO!

## Play TERMINALCASINO

To play TERMINALCASINO

1. Open a terminal (duh!)
2. Download the [GitHub repository](https://github.com/ufosc/TERMINALCASINO) onto your computer.

    ```bash
    $ git clone https://github.com/ufosc/TERMINALCASINO.git
    ```

3. Change into the `TERMINALCASINO/` directory.

    ```bash
    $ cd TERMINALCASINO/
    ```

4. Run TERMINALCASINO.

    ```bash
    $ python -m casino.main
    ```
